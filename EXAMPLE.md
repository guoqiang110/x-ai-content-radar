# Example Output Summary

Daily run creates:

- `data/raw/YYYY-MM-DD.json`: collected items
- `data/ranked/YYYY-MM-DD.md`: ranked content candidates
- `data/content-packs/YYYY-MM-DD-topic/content-pack.md`: full content pack

A content pack includes:

- Worth publishing judgment
- Xiaohongshu titles, body, tags
- 7-9 page carousel script
- Douyin/Video Account script
- AI image prompts
- Longform draft
- Publishing review checklist
